## Experiment 01: Rara Avis Game üéÆ

A fully playable **Flappy Bird clone** generated entirely by a local LLM with no external assets.

<p align="center" width="100%">
    <img width="50%" src="./experiments/01-rara-avis-game/assets/rara-avis-01.png" alt="Rara Avis gameplay">

**Highlights:**

- 100% AI-generated Python/Pygame code
- Procedural graphics (no external assets)
- Generated in seconds at 51 tokens/sec
- Model: Qwen3-Coder-30B-A3B (MLX 8-bit) - Running locally via LM Studio
